## Technologies I Used:
- HTML5: To create the structure of the pages.
- CSS3: To design the colors (I used Black and Red) and make it look good.
- Bootstrap 5: To make the website "Responsive" so it works on mobile and tablets.
- JavaScript: To add interactive things like Dark Mode and Form Validation.

## Main Features:
- Responsive Design: The website changes its layout to fit any screen size.
- Dark and Light Mode: You can switch between dark and light themes.
- Form Validation: If you forget to fill a field in the contact form, it shows an error.
- Back to Top Button: A small button to go back to the top of the page quickly.
- Navigation Menu: A simple menu that works perfectly on mobile.

## Project Pages:
- index.html: The Home page.
- about.html: Information about the gym and trainers.
- services.html: The training programs we offer.
- contact.html: How to reach us and a contact form.
- style.css: All my custom designs and colors.
- main.js: The code for the buttons and theme toggle.

## How to open it:
1. Download the files.
2. Open the index.html file in your browser.
